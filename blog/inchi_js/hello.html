InChI JavaScript demo

Enter an InChI below and click "Uninchify" to list its component atoms and bonds (or not, as the case may be). The processing is all in your browser using JavaScript. To read more about how the InChI C code was converted to JavaScript, see my blog posts [1], [2], and [3].

This only works with a recent version of Chrome (as of 21/05/2011) (i.e. don't bother trying with Firefox 4.0 or other browsers). Also, you need to refresh the page after trying an InChI.

Noel O'Boyle, May 2011.

e.g. InChI=1S/CHClO/c2-1-3/h1H
